# Contributing

Thanks for your interest in contributing!

## How to contribute

- Fork the repository
- Create a feature branch
- Make your changes
- Open a Pull Request

## Local development

### Flask (existing app)

Run:

- `python app.py`

### FastAPI rooms (new)

Run:

- `python -m uvicorn fastapi_chat.main:app --host 0.0.0.0 --port 8090 --reload`

## Pull request guidelines

- Keep changes focused and small
- Describe what you changed and why
- Include steps to test your change

## Code of Conduct

By participating, you agree to abide by `CODE_OF_CONDUCT.md`.
